# FinTech SQL Analytics Project

This project analyzes fintech transaction data to uncover insights related to customer behavior, transaction trends, merchant performance, and payment success rates.

## Project Goal
To generate data-driven insights that can support business decisions in a fintech environment.

## Project Scope
The analysis covers customer activity, transaction patterns, merchant performance, payment behavior, and risk-related signals.

## Tech Stack
- PostgreSQL
- SQL
- Git & GitHub
