# Analysis Questions

## Fraud and Risk
- Which transaction patterns appear more risky?
- Which transaction types show higher suspicious activity?
- Are high-value transactions concentrated in specific behaviors?

## Customer Behavior
- Which customers generate the highest transaction volume?
- Which account types are associated with more activity?
- Are there customer behavior patterns that indicate inactivity or risk?

## Payment Patterns
- Which payment methods are used most often?
- Which payment methods appear more reliable?
- How do payment methods differ by transaction amount?

## Business Insights
- Which transaction categories create the highest total volume?
- Are there trends across time, amount, or transaction type?
- What are the most important findings for a fintech business perspective?
